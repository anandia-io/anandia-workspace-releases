# Anandia Workspace Releases

Official releases for [Anandia Workspace](https://anandia.io) - a modern workspace application.

## Download

Download the latest version for your operating system:

| Platform | Download |
|----------|----------|
| Windows | [.msi installer](../../releases/latest) or [.exe installer](../../releases/latest) |
| macOS (Apple Silicon) | [.dmg](../../releases/latest) |
| macOS (Intel) | [.dmg](../../releases/latest) |
| Linux | [.AppImage](../../releases/latest) or [.deb](../../releases/latest) |

## Installation

### Windows
1. Download the `.msi` or `.exe` installer
2. Run the installer and follow the prompts
3. Launch Anandia Workspace from the Start menu

### macOS
1. Download the `.dmg` file
2. Open the DMG and drag Anandia Workspace to Applications
3. Launch from Applications (you may need to right-click and select "Open" the first time)

### Linux
**AppImage:**
1. Download the `.AppImage` file
2. Make it executable: `chmod +x *.AppImage`
3. Run: `./Anandia-Workspace-*.AppImage`

**Debian/Ubuntu:**
1. Download the `.deb` file
2. Install: `sudo dpkg -i *.deb`

## Auto-Updates

Anandia Workspace includes automatic updates. When a new version is available, you'll be prompted to install it.

## Verifying Downloads

All releases are signed. The `latest.json` file contains signatures for each platform that the app verifies before installing updates.
