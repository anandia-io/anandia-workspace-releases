## Features

### Infinite Canvas Workspace
- Physics-based drag & drop with spring-inertia animations
- Multi-select and batch move for organizing multiple items
- Keyboard navigation for precise positioning
- Satisfying audio feedback on interactions

### Five Item Types
- **Tasks** - Status tracking, priorities, due dates, and recurring task support
- **Habits** - Daily tracking with streaks, targets, and reminders
- **Notes** - Rich text editing with slash commands, images, and callouts
- **Projects** - Progress tracking, timelines, and status workflows
- **Folders** - Hierarchical organization with custom colors

### Smart Organization
- Nest items by dragging onto containers
- Stack mode for visual grouping
- Magnetic pull to find valid drop targets
- Sidebar tree navigation

### Local-First Architecture
- All data stored locally in SQLite - no account required
- Works completely offline
- Auto-save as you work - no save button needed

### Auto-Updates
- Automatic update checking and installation
- Signed updates for security
